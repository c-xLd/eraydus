# PURPOSE

The Admin Panel is not a traditional CMS.

It is the operational control center of the ERAYDUŞ Digital Experience Platform.

Every product, price, configuration rule, quotation request, blog article and SEO setting must be manageable without writing code.

The interface should feel closer to:

- Shopify Admin
- Stripe Dashboard
- Vercel Dashboard
- Linear
- Notion

Rather than a typical admin panel.
