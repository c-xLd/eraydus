# PURPOSE

This document defines every reusable UI component used throughout the ERAYDUŞ Digital Experience Platform.

Every component must be reusable.

Every component must support Light Mode and Dark Mode.

Every component must be fully responsive.

Every component must support accessibility (WCAG AA).

Every component must follow the Premium Design System.

No page may create its own custom component unless absolutely necessary.
